- **Asset collection** - Multiple themed resources
- **Visual reference** - PDF showcase of themes
- **Structured data** - Consistent theme format
- **Use case guidance** - When to use each theme
- **Accessibility focus** - Contrast and readability notes

## Restoration

To recreate the theme resources:
1. Create 10 theme markdown files in `themes/` directory
2. For each theme, define 5-7 colors with:
   - Hex codes and RGB values
   - Usage guidelines (headings, body, accents)
   - Contrast ratios for accessibility
   - Typography recommendations
   - Use case examples
3. Create `theme-showcase.pdf` with visual examples of each theme
4. Test themes in actual presentations/documents

See SKILL.md for the complete skill implementation with inline theme examples.
