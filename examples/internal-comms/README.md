## Removed Resources (Described Below)

### examples/

This skill originally included example outputs for different communication types. These served as reference implementations showing tone, structure, and formatting for various internal communications.

#### 3p-updates.md
Example of third-party vendor updates and communications:
- Vendor change notifications
- Service updates and migrations
- Contract renewal communications
- Partner relationship updates
- Integration status reports

**Tone:** Professional, informative, business-focused  
**Length:** 200-400 words  
**Format:** Structured with headers, bullet points, action items

#### company-newsletter.md
Example company-wide newsletter:
- Company news and achievements
- Team spotlights and celebrations
- Product updates and milestones
- Upcoming events and initiatives
- Employee resources and benefits

**Tone:** Engaging, positive, inclusive  
**Length:** 500-800 words  
**Format:** Multiple sections, visual breaks, links

####faq-answers.md
Example FAQ responses for common questions:
- Policy clarifications
- Process explanations
- Benefit details
- Tool usage guidance
- Common troubleshooting

**Tone:** Clear, helpful, empathetic  
**Length:** 100-200 words per Q&A  
**Format:** Q&A pairs, step-by-step when applicable

#### general-comms.md
Example general announcements and updates:
- Office updates and changes
- Policy announcements
- Schedule changes
- General reminders
- Process updates

**Tone:** Professional, clear, concise  
**Length:** 150-300 words  
**Format:** Clear structure, action items highlighted
